# Adrià Molina Hernandez - Portfolio

Portfolio profesional estático compatible con GitHub Pages.

## Despliegue en GitHub Pages

1. Crea un repositorio en GitHub
2. Sube todos los archivos del proyecto
3. Ve a Settings > Pages
4. Selecciona la rama principal (main/master) como fuente
5. Tu sitio estará disponible en `https://tu-usuario.github.io/nombre-repositorio`

## Estructura

- `index.html` - Página principal
- `styles.css` - Estilos CSS
- `script.js` - Lógica JavaScript
- `images/` - Carpeta para todas las imágenes y videos
- `icon.svg` - Favicon del sitio

## Características

- Diseño totalmente responsive
- Modo oscuro/claro
- Soporte bilingüe (Español/Inglés)
- Animaciones suaves
- Compatible con todos los navegadores modernos
- Sin dependencias externas
